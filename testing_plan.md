# 📋 План Тестирования и Проверки

Этот план поможет убедиться, что каждый компонент системы работает идеально перед запуском на реальном магазине.

## 1. Проверка Подключения и Профиля
- [ ] **Вход через Telegram**: Откройте Mini App. Убедитесь, что ваш Telegram ID определился верно.
- [ ] **Авто-регистрация**: Проверьте в Supabase, что в таблице `sellers` появилась запись с вашим ID.
- [ ] **Статус Premium**: Первые 5 пользователей должны получить статус `premium` автоматически.

## 2. Настройки и Инструкции
- [ ] **Сохранение токена**: Введите `MOCK_TOKEN`. Убедитесь, что он сохранился.
- [ ] **Tone of Voice**: Введите кастомную инструкцию (например, "Отвечай как ковбой").

## 3. Матрица Товаров (Cross-sell)
- [ ] **Добавление пары**: Добавьте связку: Артикул `123456` -> Рекомендация `987654`.

## 4. Сценарии ИИ (Главный тест)
Используйте скрипт `node run_scenarios.js` для имитации следующих ситуаций:
- [ ] **Позитив (5 звезд)**: Должен быть вежливый ответ + рекомендация из матрицы.
- [ ] **Негатив (1-2 звезды)**: Эмпатичный ответ без рекомендации.

## 5. Бот-Консультант (Поддержка)
- [ ] **Память**: Спросите бота "Кто ты?", а затем "А что еще ты умеешь?".
- [ ] **Эскалация**: Напишите "Свяжи с админом".

## 6. Автоматизация (Cron)
- [ ] **Ручной запуск**: Нажмите кнопку "Синхронизировать отзывы".
- [ ] **GitHub Actions**: Проверьте вкладку "Actions" на GitHub.
